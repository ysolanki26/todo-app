# 📝 Smart To-Do List App

## 🚀 Features
- ➕ Add tasks
- ✔️ Mark as complete
- ❌ Delete tasks
- 🗑️ Clear all tasks
- 🎨 Beautiful UI with emojis

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS

## ▶️ How to Run
1. Install Flask:
   pip install flask

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000/
